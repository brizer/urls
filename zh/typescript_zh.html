# Typescript

# 文档

  • TypeDoc - 为typescript项目生成文档

# 例子

  • domutils - DOM util，基于typescript
  • google-api-nodejs-client - 谷歌官方api的node版本客户端，由typescript完成。
  • jira_clone - 类似jira的工具，基于typescript/express/react/reacthook/postgres
  • zan-proxy - 有赞请求代理工具，客户端由electron/typescript
  • react-antd-admin - 后台前端管理系统，基于react hooks、typescript、antd、dva及一些特别优秀的开源库实现
  • react-use - 一大把react hooks的最佳实践和demo，基于typescript
  • kutt - 短链服务，后台由Node/Express/Typescript/PostgreSQL构建

# 命令行

  • tsdx - 0配置开箱即用的ts包开发工具

# 其他

  • DefinitelyTyped - 针对第三方库的typescript类型定义
  • mini-typescript - 迷你版tsc的实现，可以用来了解typescript编译过程
    • 源码分析
  • ts-morph - 解析ts为抽象语法树
  • ts-node - Typescript 执行器
    • 源码分析
  • tslib - ts的helper运行时库。
  • typed-css-module - 为css生成 .d.ts文件，方便css module
  • types-publisher - 将DefinitelyTyped的内容推到npm的服务
  • TypeScript - TypeScript是javascript的超级，编译纯净的js输出。
    • 源码分析
  • typescript-book - 关于typescript最好的书
