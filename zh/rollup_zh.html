# Rollup

  • awesome - 社区资源总集

# 项目实例

  • furl - 将多个react组件打包到一个输出
  • rollup-plugin-babel - 一个rollup插件，打包ts为cjs和esm文件。
  • rollup-react-nej - 将.jsx文件转为amd风格的.js文件，并使得nej调用成功。
  • rollup-vue-nej - 将.vue文件转为amd风格的.js文件，并使得nej调用成功。
  • vue-runtime-helpers - 从src文件内所有编译为dist下

# 其他

  • @rollup/plugin-alias - 一个定义别名aliases的rollup插件
  • @rollup/plugin-json - 将json文件转为es6的模块
  • rollup - rollup本体
    • 源码分析
  • rollup-plugin-amd - 将amd转换为es6模块
  • rollup-plugin-auto-external - 打包时自动排除package.json中的依赖
  • rollup-plugin-babel - 在rollup中使用babel
  • rollup-plugin-cleanup - 移除注释、空行、空格缩进等
  • rollup-plugin-commonjs - 将commonjs转换为es6模块
  • rollup-plugin-html - 将html文件作为字符串引入
  • rollup-plugin-license - 增加开源协议头
    • 被sentry-js-sdk使用
  • rollup-plugin-livereload - 热重载打包文件
  • rollup-plugin-node-resolve - 通过node找模块算法来找模块，用于加载处于node_modules中的第三方模块
    • 被vue ui使用
  • rollup-plugin-postcss - 加载css，less，scss的插件
  • rollup-plugin-replace - 替换文件中的字符串
    • 被vue ui使用
  • rollup-plugin-serve - 类似webpack-dev-server一样的启动rollup打包文件
  • rollup-plugin-string - 加载文本字符串模块
  • rollup-plugin-terser - 压缩生成的es包
  • rollup-plugin-typescript - rollup的typescript插件
  • rollup-plugin-typescript2 - rollup的typescript插件，带有编译报错
  • rollup-plugin-vue - 加载vue文件
