# loader

  • babel-loader - babel的loader
    • 简单使用babel和jsx
  • cache-loader - 可以在一些性能开销较大的 Loader 之前添加，目的是将结果缓存到磁盘里
  • css-loader - 支持引入css
  • ejs-loader - .ejs文件的loader
  • eslint-loader - eslint的loader
  • file-loader - png等图片文件的loader
  • html-loader - HTML文件的loader
  • less-loader - 针对less的loader
    • 简单的使用less
    • 和cssmodule,react一起使用
  • postcss-loader - PostCSS的loader
  • raw-loader - txt文件转为字符串的loader
    • 源码分析，最小的loader，适合入门
  • style-loader - 通过style标签将css加到dom中去
  • svg-sprite-loader - svg图标切割loader，用来更加方便的使用svg图标
  • ts-loader - typescript的loader
  • url-loader - 转换图片文件到base64
  • vue-loader - vue文件的loader

# plugins

  • clean-webpack-plugin - 目录清理
  • copy-webpack-plugin - 复制文件到生成目录
  • create-file-webpack - 在webpack打包过程中创建文件内容
    • 源码分析，最小的插件，适合入门
    • [可用于生成网站地图sitemap]
  • hard-source-webpack-plugin - 缓存文件，优化构建速度
    • 源码分析
  • html-webpack-plugin - 简化html创建和启动服务的插件
  • mini-css-extract-plugin - 分离样式文件，CSS 提取为独立文件，css分离插件
  • offline-package-webpack-plugin - 将打包后的文件压缩到一个zip中，并将文件内容生成json
  • 源码分析
  • sentry-webpack-plugin - Sentry官方的webpack插件 https://sentry.io
  • size-plugin - ：监控资源体积变化，尽早发现问题
  • speed-measure-webpack-plugin#readme - 记录各个loader和插件的耗时，方便排查构建速度
  • terser-webpack-plugin - js最小化、多进程开启
  • webpack-bundle-analyzer - 可视化分析webpack依赖模块大小
  • webpack-deadcode-plugin - 分析webpack打包的文件中，哪些文件是死代码，不需要被依赖进去
  • webpack-manifest-plugin - 生成静态资源manifest的插件
  • webpack-subresource-integrity - 使用Subresource Integrity的webbpack插件
  • write-file-webpack-plugin - 使得webpack-dev-server将打包文件写到文件系统中
