# Babel

  • @babel/core - Babel编译核心
    • babel生态及源码分析
  • @babel/generator - 将AST转换成js代码，内置流程的工具
  • @babel/node - Babel的node命令行工具，使用babel的配置来执行nodejs文件
  • @babel/polyfill - 浏览器的兼容性
  • @babel/register - 通过API给require增加处理钩子
    • 被umi使用
  • @babel/runtime - babel模块的运行时帮助函数集
    • 为什么需要runtime
  • babel-bridge - 联通babel-core到@babel/core。 如果你使用的某个包需要babel-core，而你却使用的@babel/core，则你需要它。
  • babel-eslint - 适配eslint和babel
  • babel-jest - Babel的jest插件

# 插件

  • @babel/plugin-proposal-class-properties - 使用class的属性语法
  • @babel/plugin-transform-function-name - 使用function.name语法的插件
  • @babel/plugin-transform-modules-commonjs - 将ES2015的模块转为commonjs模块
  • babel-plugin-bulk-import - glob语法import多文件
  • babel-plugin-import - 动态加载antd
  • babel-plugin-lodash - lodash拆分引入的插件，lodash官方提供
  • babel-plugin-minify-replace - 转换某些字符串
  • babel-plugin-transform-class-properties - 支持class的属性类似属性初始化语法
  • babel-plugin-transform-vue-jsx - 启用vue的jsx语法

# 预设集

  • @babel/preset-env - 各个环境的预设集
  • @babel/preset-react - React 的预设集
  • @babel/preset-typescript - Typescript 的预设集
