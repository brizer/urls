# react native

# 实例

  • ignite - 比较火的RN模板示例
  • MetroExample - 一个基于metro拆包比较好的实践
  • react-native-example - 一些react-native的例子，brizer开发
  • react-native-multibundler - react native可视化bundle拆包,支持远程加载和debug,支持官方的0.57~0.63.2版本,使用官方的metro拆包,适用于Android、iOS

# 动画

  • lottie - rn、web动画工具，非常方便和通用

# 工具

  • expo-cli - 方便的查看工具，react-native默认cli，简介方便
  • flipper - Facebook出品的桌面调试工具，RN可以用
  • react-native-bundle-visualizer - 查看rn打包文件大小
  • react-native-cli - react-native官方cli，需要xcode和android studios.
  • react-native-web - 将react-native组件转为web运行

# 组件

  • react-native-lightbox - 全屏蒙层查看器，适合图片查看详情

# 组件库

  • ant-design-mobile-rn - 蚂蚁金服组件库
  • NativeBase - Mobile-first, accessible components for React Native & Web to build a customizable Design System.
  • react-native-elements - 评分非常高的组件库

# 图片相关

  • react-native-fast-image - 网络图片缓存库
  • react-native-view-shot - 对比html2canvas的海报库

# Util

  • async-storage - 异步存储库，支持native和h5
  • react-native-cameraroll - 读取相册和图片库
