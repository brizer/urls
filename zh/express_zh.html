# Express

# 项目实例

  • AccountSystem - express/mongoose的库存系统
  • CNode - CNode 源码
  • jira_clone - 类似jira的工具，基于typescript/express/react/reacthook/postgres
  • kite - 🌴 Kite 前台页面是vue ssr服务端渲染、后台页面是react spa、服务层nodejs、mysql编写的一套多权限内容管理系统
  • kutt - 短链服务，后台由Node/Express/Typescript/PostgreSQL构建
  • proshop_mern - 购物车，基于react/redux/mongo/express
  • vueMusic - h5音乐软件基于vue/docker/docker-compose/express

# 中间件

  • body-parser - 解析body的中间件
    • 在一个简单crud项目中使用
  • cookie-parser - 解析http请求的cookies
    • 在一个简单crud项目中使用
  • cookie-session - 简单的 基于cookie的 session 中间件
  • express-fileupload - 文件上传处理中间件
  • express-healthcheck - 极简监控检查中间件
  • express-rate-limit - 速率限制中间件
  • express-velocity - 渲染velocity模板
  • helmet - 通过各种http头保护express应用
    • 描述它的文章
  • method-override - 复写客户端不支持的http的method如put/delete等，中间件
  • morgan - http请求日志中间件
  • serve-favicon - favicon图标处理中间件
    • 在一个简单crud项目中使用
  • serve-static - 静态文件服务中间件
  • session - session中间件

# 其他

  • express - express 框架本身

    • express源码分析

  • express-interceptor - Express的响应拦截器(弥补非洋葱模型)

  • express-mung - Express的响应转化器(弥补非洋葱模型)

  • SwaggerUIExpress - 一个express中间件，将swagger文档ui加入
