# ESLint

  • eslint - Check the es writing specification
  • eslint-config-airbnb - Aribnb's javascript style
  • eslint-config-prettier - turns off conflict rules with Prettier
  • eslint-plugin-babel - companion to babel-eslint
  • eslint-plugin-compat - Lint the browser compatibility
  • eslint-plugin-import - validate proper imports
  • eslint-plugin-jsx-a11y - JSX elements checker
  • eslint-plugin-local - import local plugins
  • eslint-plugin-markdown - lint js in md
  • eslint-plugin-react - lint for react
  • eslint-plugin-vue - lint for vue

# StyleLint

  • stylelint - style linter
  • stylelint-config-css-modules - css modules linter
  • stylelint-config-prettier - turns off conflict rules with Prettier
  • stylelint-config-rational-order - lint css property orders
  • stylelint-config-standard - standard config
