# Babel

  • @babel/core - Babel compiler core
    • babel source analysis
  • @babel/generator - transform AST to js code
  • @babel/node - Babel cli for node, use babel config to exec node js
  • @babel/polyfill - Browser compatibility
  • @babel/register - add hook for require
    • used by umi
  • @babel/runtime - babel's modular runtime helpers
    • Why we need it?
  • babel-bridge - A placeholder package that bridges babel-core to @babel/core. If you use some package that need babel-core, bu you use @babel/core, then you need it.
  • babel-eslint - wrapper for babel's parser used for ESLint
  • babel-jest - Babel jest plugin

# plugins

  • @babel/plugin-proposal-class-properties - use properties in class
  • @babel/plugin-transform-function-name - Apply ES2015 function.name semantics to all functions
  • @babel/plugin-transform-modules-commonjs - This plugin transforms ES2015 modules to CommonJS.
  • babel-plugin-bulk-import - Babel plugin for bulk import
  • babel-plugin-import - import antd dynamic
  • babel-plugin-lodash - Modular Lodash builds without the hassle.
  • babel-plugin-minify-replace - replace some string
  • babel-plugin-transform-class-properties - support class properties as well as property initializer
  • babel-plugin-transform-vue-jsx - babel plugin for vue 2.0 jsx

# presets

  • @babel/preset-env - preset of each environment
  • @babel/preset-react - preset of React
  • @babel/preset-typescript - preset of Typescript
