# plugins

  • clean-webpack-plugin - A webpack plugin to remove your build folder(s) before building
  • copy-webpack-plugin - Copy files to build directory
  • create-file-webpack - The simple webpack plugin to create a file with the particular content at the end of the build process
    • source analysis

  • hard-source-webpack-plugin - cache file to reduce the time of building
    • source analysis
  • html-webpack-plugin - Plugin that simplifies creation of HTML files to serve your bundles
  • mini-css-extract-plugin - Lightweight CSS extraction plugin
  • offline-package-webpack-plugin - Webpack plugin helps compress static resources (such as js, css, png...) into a zip package, with a resource mapping json file in it.
  • source analysis
  • sentry-webpack-plugin - Official webpack plugin for Sentry https://sentry.io
  • size-plugin - Track compressed Webpack asset sizes over time.
  • speed-measure-webpack-plugin#readme - ‚è± See how fast (or not) your plugins and loaders are, so you can optimise your builds
  • terser-webpack-plugin - Terser Plugin
